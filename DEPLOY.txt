TROUBLESHOOTING
================================================================================

Blank Page:
- Check all 3 files uploaded
- Wait 5 minutes
- Clear browser cache

404 Error:
- Check GitHub Pages enabled
- Check branch is "main"
- Check folder is "/ (root)"

Not Responsive:
- Make sure styles.css uploaded
- Check file names match exactly
